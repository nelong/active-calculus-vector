Activity 10.2.3.

  1. If \(f(x,y) = 3x^3 - 2x^2y^5\text{,}\) find the partial derivatives \(f_x\) and \(f_y\text{.}\)
  2. If \(f(x,y) = \displaystyle\frac{xy^2}{x+1}\text{,}\) find the partial derivatives \(f_x\) and \(f_y\text{.}\)
  3. If \(g(r,s) = rs\cos(r)\text{,}\) find the partial derivatives \(g_r\) and \(g_s\text{.}\)
  4. Assuming \(f(w,x,y) = (6w+1)\cos(3x^2+4xy^3+y)\text{,}\) find the partial derivatives \(f_w\text{,}\) \(f_x\text{,}\) and \(f_y\text{.}\)
  5. Find all possible first-order partial derivatives of \(q(x,t,z) = \displaystyle \frac{x2^tz^3}{1+x^2}.\)
