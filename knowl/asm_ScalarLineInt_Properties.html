Properties of Scalar Line Integrals.

For a constant scalar \(k\text{,}\) scalar valued functions\(f\) and \(g\text{,}\) and oriented curves \(C\text{,}\) \(C_1\text{,}\) and \(C_2\text{,}\) the following properties hold:
  1. \(\displaystyle \displaystyle \int_C (k f) \enspace ds = k \int_C f \enspace ds\)
  2. \(\displaystyle \displaystyle \int_C (f+g) \enspace ds = \int_C f \enspace ds + \int_C g \enspace ds \)
  3. \(\displaystyle \displaystyle \int_{-C} f \enspace ds = \int_C f \enspace ds\)
  4. \(\displaystyle \displaystyle \int_{C_1+C_2} f \enspace ds = \int_{C_1} f \enspace ds + \int_{C_2} f \enspace ds\)
