Sketch the region of integration for the following integral.

\(\displaystyle \int_{0}^{\pi/4} \int_{0}^{6 / \cos(\theta)} f(r,\theta) \, r \, dr \, d\theta\)

The region of integration is bounded by

  • \(\displaystyle y = 0, x = \sqrt{36 - y^2}, \mbox{ and } y = 6\)

  • \(\displaystyle y = 0, y = x, \mbox{ and } x = 6\)

  • \(\displaystyle y = 0, y = x, \mbox{ and } y = 6\)

  • \(\displaystyle y = 0, y = \sqrt{36 - x^2}, \mbox{ and } x = 6\)

  • None of the above
