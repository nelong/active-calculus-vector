Example 12.4.2.

Suppose we need to calculate the line integral of \(\vF= \langle 2xy-2, x^2+3e^y \rangle\) over the path from \((2,0)\) to \((4,-2)\) along the curve given by \((x-2)^2=-y/2\text{.}\) If we use Theorem 12.3.5 to solve this, we would need to parameterize our curve, then take a derivative of the parameterization, then substitue into the line integral formula, then integrate the result.
In order to use Theorem 12.4.1, we first need to verify that there exists a potential function \(f(x,y)\) such that \(\nabla f = \langle 2xy-2, x^2+3e^y \rangle\text{.}\) You can use the techniques of Activity 12.4.2 to find that \(f\) will be of the form \(x^2y-2x-3e^y+k\text{.}\) Note that \(f(4,-2)- f(2,0)= (-32-8+3e^{-2}+k)-(0-4-3+k)= 3e^{-2}-41\text{,}\) regardless of what value you choose for \(k\text{.}\) By Theorem 12.4.1,
\begin{equation*} \int_C \vF \cdot d\vr = f(4,-2)- f(2,0)= 3e^{-2}-41 \end{equation*}
If you changed the path in your line integral to be any other path between \((2,0)\) and \((4,-2)\text{,}\) then the result would still be \(3e^{-2}-41\text{.}\) If you changed the path in your line integral to be some path from \((1,-1)\) to \((-1,4)\text{,}\) then you would only need to evaluate \(f(-1,4)- f(1,-1)\text{.}\) So for our new path \(C_1\text{,}\)
\begin{equation*} \int_{C_1} \vF \cdot d\vr = f(-1,4)- f(1,-1) = (4-(-2)+3e^4)-(-1-2+3e^{-1})=9+3(e^4+e^{-1}) \end{equation*}
