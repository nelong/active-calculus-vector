Exercise 12.

The temperature at any point on a metal plate in the \(xy\)-plane is given by \(T(x,y) = 100-4x^2 - y^2\text{,}\) where \(x\) and \(y\) are measured in inches and \(T\) in degrees Celsius. Consider the portion of the plate that lies on the region \(D\) that is the finite region that lies between the parabolas \(x = y^2\) and \(x = 3 - 2y^2\text{.}\)

  1. Construct a labeled sketch of the region \(D\text{.}\)

  2. Set up an iterated integral whose value is \(\iint_D T(x,y) \, dA\text{,}\) using \(dA = dx dy\text{.}\) (Hint: It is possible that more than one integral is needed.)

  3. Set up an integrated integral whose value is \(\iint_D T(x,y) \, dA\text{,}\) using \(dA = dy dx\text{.}\) (Hint: It is possible that more than one integral is needed.)

  4. Use the Fundamental Theorem of Calculus to evaluate the integrals you determined in (b) and (c).

  5. Determine the exact average temperature, \(T_{\operatorname{AVG}(D)}\text{,}\) over the region \(D\text{.}\)
