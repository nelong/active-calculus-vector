Equation (9.7.1) for the tangent line \(\vL(t)\) to the vector-valued function \(\vr(t)\) is nearly identical. Indeed, because there are multiple parameterizations for a single line, it is even possible to write the parameterization as
\begin{equation} \vL(t) = \vr(a) + (t-a) \vr'(a).\tag{9.7.2} \end{equation}
(For example, in Equation (9.7.1), \(\vL(0) = \vr(a)\text{,}\) so the line’s parameterization “starts” at \(t = 0\text{.}\) When we write the parameterization in the form of Equation (9.7.2), \(\vL(a) = \vr(a)\text{,}\) so the line’s parameterization “starts” at \(t = a\text{.}\))
