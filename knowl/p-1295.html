Using an arc length parametrization \(\vr(s)\text{,}\) we define the tangent vector \(\vT(s) = \vr'(s)\text{,}\) and note that \(|\vT(s)| = 1\text{;}\) that is, \(\vT(s)\) is a unit tangent vector. We then have \(\vT(s) = \langle \cos (\phi(s)), \sin(\phi(s)) \rangle\text{,}\) which means that
\begin{equation*} \frac{d\vT}{ds} = \left\langle -\sin(\phi(s)) \frac{d\phi}{ds}, ~ \cos(\phi(s)) \frac{d\phi}{ds} \right\rangle = \langle -\sin(\phi(s)),~ \cos(\phi(s)) \rangle \frac{d\phi}{ds}. \end{equation*}
