  1. If \(\vec{u}\) is a unit vector, then \(f_{\vec{u}} (a,b)\) is a vector.
  2. If \(\vec{u}\) is perpendicular to \(\nabla f(a,b)\text{,}\) then \(f_{\vec{u}} \, (a,b) = \langle 0, 0 \rangle\text{.}\)
  3. Suppose \(f_x(a,b)\) and \(f_y(a,b)\) both exist. Then there is always a direction in which the rate of change of \(f\) at \((a,b)\) is zero.
  4. If \(f(x,y)\) has \(f_x(a,b) = 0\) and \(f_y(a,b) = 0\) at the point \((a,b)\text{,}\) then \(f\) is constant everywhere.
  5. \(\nabla f(a,b)\) is a vector in 3-dimensional space.
  6. \(f_{\vec{u}} \, (a,b) = || \nabla f(a,b) ||\text{.}\)
  7. The gradient vector \(\nabla f(a,b)\) is tangent to the contour of \(f\) at \((a,b)\text{.}\)
  8. \(f_{\vec{u}} \, (a,b)\) is parallel to \(\vec{u}\text{.}\)
