For a constant scalar \(k\text{,}\) vector fields \(\vF\) and \(\vG\text{,}\) and oriented curves \(C\text{,}\) \(C_1\text{,}\) and \(C_2\text{,}\) the following properties hold:
  1. \(\displaystyle \displaystyle\int_C (k\vF)\cdot d\vr = k\int_C\vF\cdot d\vr\)
  2. \(\displaystyle \displaystyle\int_C(\vF+\vG)\cdot d\vr = \int_C \vF\cdot d\vr + \int_C\vG\cdot d\vr\)
  3. \(\displaystyle \displaystyle\int_{-C}\vF\cdot d\vr = -\int_C \vF\cdot d\vr\)
  4. \(\displaystyle\int_{C_1+C_2} \vF\cdot d\vr = \int_{C_1}\vF\cdot d\vr + \int_{C_2}\vF\cdot d\vr\text{.}\)
