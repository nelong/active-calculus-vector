If you changed the path in your line integral to be any other path between \((2,0)\) and \((4,-2)\text{,}\) then the result would still be \(3e^{-2}-41\text{.}\) If you changed the path in your line integral to be some path from \((1,-1)\) to \((-1,4)\text{,}\) then you would only need to evaluate \(f(-1,4)- f(1,-1)\text{.}\) So for our new path \(C_1\text{,}\)
\begin{equation*} \int_{C_1} \vF \cdot d\vr = f(-1,4)- f(1,-1) = (4-(-2)+3e^4)-(-1-2+3e^{-1})=9+3(e^4+e^{-1}) \end{equation*}
