Recall that in single variable calculus, the Second Fundamental Theorem of Calculus tells us that given a constant \(c\) and a continuous function \(f\text{,}\) there is a unique function \(A(x)\) for which \(A(c) = 0\) and \(\frac{dA}{dx} (x) = f(x)\text{.}\) In particular, \(A(x) = \int_c^x f(t)\, dt\) is this function. We are about to investigate an analog of this result for path-independent vector fields, but first we require two additional definitions.
