We parameterize \(C_1\) with \(\vr_1(t)= \langle -1=t,2-2t\rangle\) and parameterize \(C_2\) with \(\vr_2(t) =\langle t,t\rangle \text{.}\) Both parameterizations use parameter values in the interval \([0,1]\text{.}\) We can calculate \(\Vert{\vr_1}\Vert = \sqrt{5}\) and \(\Vert{\vr_2}\Vert = \sqrt{2}\text{.}\) Thus we can evaluate our scalar line integral as
\begin{gather*} \int_{C_1+C_2} x \enspace ds = \int_{C_1} x \enspace ds +\int_{C_2} x \enspace ds\\ \quad = \int_0^1 (-1+t) \sqrt{5} \enspace dt + \int_0^1 (t) \sqrt{2} \enspace dt\\ \quat = -\sqrt{5}+\frac{\sqrt{5}}{2}+\frac{\sqrt{2}}{2} \neq 0 \end{gather*}
In the calculation above, you can see how the difference in displacements manifests differently in the two integrals (as \(\sqrt{5}\) and \(\sqrt{2}\)) which makes the final result not quite cancel out to zero.
