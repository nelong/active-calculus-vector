We can parametrize the top edge of the box (in blue) by \(\vr_{\text{top}}(t) = \langle a+t,b+h\rangle\) with \(-h\leq t\leq h\text{.}\) Similarly, the bottom, right, and left can be parametrized by
  • \(\displaystyle \vr_{\text{bottom}}(t) = \langle a+t,b-h\rangle\)
  • \(\displaystyle \vr_{\text{right}}(t) = \langle a+h,b+t\rangle\)
  • \(\displaystyle \vr_{\text{left}}(t) = \langle a-h,b+t\rangle\)
all of which use the parameter values in \(-h\leq t\leq h\text{.}\)
