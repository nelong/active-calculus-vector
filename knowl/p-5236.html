For a constant scalar \(k\text{,}\) scalar valued functions\(f\) and \(g\text{,}\) and oriented surfaces \(S_1\)and \(S_2\text{,}\) the following properties hold:

  1. \(\displaystyle \displaystyle \iint_{S_1} (k f) \enspace dS = k \iint_{S_1} f \enspace dS\)

  2. \(\displaystyle \displaystyle\iint_{S_1} (f+g) \enspace dS = \iint_{S_1} f dS + \iint_{S_1} g \enspace dS \)

  3. \(\displaystyle \displaystyle\iint_{-S_1} f \enspace dS = \iint_{S_1} f \enspace dS\)

  4. \(\displaystyle \displaystyle\iint_{S_1+S_2} f \enspace dS = \iint_{S_1} f \enspace dS + \iint_{S_2} f \enspace dS\)
