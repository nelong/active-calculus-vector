We will need to compute the line integral of \(\vF\) over the four segements of the bounding rectangle. We will use the following parameterizations:
  • \(\displaystyle \vr_{bottom}(t) = \langle 1+3t,2 \rangle \)
  • \(\displaystyle \vr_{right}(t) = \langle 4,2+2t \rangle \)
  • \(\displaystyle \vr_{top}(t) = \langle 4-3t,4 \rangle \)
  • \(\displaystyle \vr_{left}(t) = \langle 1,4-2t \rangle \)
where each paramterization uses \(0\leq t\leq 1\text{.}\) These parameterizations will yeild the following line integrals:
  • \(\displaystyle \int_{bottom} \vF \cdot d\vr = \int_0^1 ( (1+3t)^2 -(1+3t)(2) ) 3 \enspace dt = 6 \)
  • \(\displaystyle \int_{right} \vF \cdot d\vr = \int_0^1 ( (2+2t)^2 -(2)(4) ) 2 \enspace dt = 8/3 \)
  • \(\displaystyle \int_{top} \vF \cdot d\vr = \int_0^1 ( (4-3t)^2 -(4-3t)(4) ) (-3) \enspace dt = 9 \)
  • \(\displaystyle \int_{left} \vF \cdot d\vr = \int_0^1 ( (4-2t)^2 -(2)(1) ) (-2) \enspace dt = -44/3 \)
Note that the sum of these line integrals is 3, and thus Greenâ€™s Theorem is verified for our case since
\begin{equation*} \oint_C \vF\cdot d\vr = 3 = \iint_R \left(\frac{\partial F_2}{\partial x} - \frac{\partial F_1}{\partial y}\right)\, dA \end{equation*}
