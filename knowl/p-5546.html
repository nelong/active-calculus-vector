If we want to calculate the integral of \(f(x,y,z) = x\) over \(S_2\) given by \(x^2+y^2=z^2\) such that \(1\leq z \leq 3\) and \(0\leq x \text{,}\) then we have very little extra work to do. Only the bounds on the integral will change since we are using the same parameterization and function from the first example. We can use the parameterization given by \(\vr(s,t)=\langle s \cos(t), s \sin(t), s \rangle \) with \(1\leq s \leq 3\) and \(-\frac{\pi}{2} \leq t \lt \frac{\pi}{2} \text{.}\) The length of \(\vw\) will be \(\sqrt{s^2(\cos(t)^2+\sin(t)^2) +s^2}=2s\text{.}\) Thus, TheoremÂ 12.10.4 gives
