Now we are ready to set up the flux integral as the following iterated integral:
\begin{equation*} \int_0^1 \int_0^{2\pi} \langle s\cos(t)-1,1-s\sin(t),0 \rangle \cdot \langle -2s^2 \cos(t), 2s^2 \sin(t), s \rangle \enspace dt ds \end{equation*}
You probably noticed that this integral will look more slightly more complicated than our work in the previous part, but if we are consistent, we should get the same result. Evaluating this dot product, then computing each integral gives:
\begin{gather*} \int_0^1 \int_0^{2\pi} (-2s^2 \cos(t))(s\cos(t)-1) + (2s^2 \sin(t))(1-s\sin(t)) \enspace dt ds \\ \int_0^1 \int_0^{2\pi} -2s^3 \cos(t)^2 + 2s^2 \cos(t) + 2s^2 \sin(t) -2s^3\sin(t)^2 \enspace dt ds \\ \int_0^1 \int_0^{2\pi} -2s^3 + 2s^2 \cos(t) + 2s^2 \sin(t) \enspace dt ds \end{gather*}
Because both sine and cosine will integrate to zero over the interval from zero to \(2\pi\text{,}\) we only need to evaluate
\begin{equation*} 2 \pi \int_0^1 -2s^3 \enspace ds= -\pi \end{equation*}
which gives exactly the same result as our circulation integrate AND our flux integral with the other surface.
