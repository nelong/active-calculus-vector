\((x,y) = \left(2,-3\right)\) to find \(f(2,-3) = -3 + 2(-3)^{5} = -489\text{.}\) Similarly, with \((x,y) = \left(3,-1\right)\) we find \(f(3,-1) = -1 + 3(-1)^{5} = -4\text{.}\) Finally, for \((x,y) = \left(-5,-2\right)\) we find \(f(-5,-2) = -2 - 5(-2)^{5} = 158\text{.}\)
