Solution 12.5.3.c.1.
We will use TheoremÂ 12.5.11 to evaluate our line integral. We parameterize \(C_1\) with \(\vr_1(t)= \langle -1=t,2-2t\rangle\) and parameterize \(C_2\) with \(\vr_2(t) =\langle t,t\rangle \text{.}\) Both parameterizations use parameter values in the interval \([0,1]\text{.}\) We can calculate \(|{\vr_1}| = \sqrt{5}\) and \(|{\vr_2}| = \sqrt{2}\text{.}\) Thus we can evaluate our scalar line integral as
\begin{align*} \int_{C_1+C_2} x \, ds \amp = \int_{C_1} x \, ds +\int_{C_2} x \, ds\\ \amp = \int_0^1 (-1+t) \sqrt{5} \, dt + \int_0^1 (t) \sqrt{2} \, dt\\ \amp = -\sqrt{5}+\frac{\sqrt{5}}{2}+\frac{\sqrt{2}}{2} \neq 0 \end{align*}
In the calculation above, you can see how the difference in displacements manifests differently in the two integrals (as \(\sqrt{5}\) and \(\sqrt{2}\)) which makes the final result not quite cancel out to zero.
