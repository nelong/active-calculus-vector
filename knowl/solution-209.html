Solution 12.7.6.6.1.

(a) Since \(\mbox{curl } \vec F = \boldsymbol{0}\)and \(\vec F\) is defined everywhere, we know by the curl test that \(\vec F\) is a gradient field. (b) Since \(\mbox{curl } \vec G = -2z\,\mathit{\vec i}+\left(y-4z\right)\,\mathit{\vec j}-\left(x+z\right)\,\mathit{\vec k}\) we know by the curl test that \(\vec G\) is not a gradient field. (c) Since \(\mbox{curl } \vec H = \boldsymbol{0}\)and \(\vec H\) is defined everywhere we know by the curl test that \(\vec H\) is a gradient field.
