Solution 12.7.6.4.1.

(a) Since \(\mbox{curl } \vec F = \boldsymbol{0}\)and \(\vec F\) is defined everywhere, we know by the curl test that \(\vec F\) is a gradient field. (b) Since \(\mbox{curl } \vec G = \boldsymbol{0}\)and \(\vec G\) is defined everywhere we know by the curl test that \(\vec G\) is a gradient field. (c) Since \(\mbox{curl } \vec H = 8y\,\mathit{\vec i}+12x\,\mathit{\vec k}\) we know by the curl test that \(\vec H\) is not a gradient field.
