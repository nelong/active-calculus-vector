Solution 12.10.3.2.1.

\begin{equation*} \mbox{curl }\vec F = \left| \begin{array}{ccc} \vec i \amp \vec j \amp \vec k\\ \frac{\partial}{\partial x}\ \amp \ \frac{\partial}{\partial y} \ \amp \ \frac{\partial}{\partial z}\\ 5y \amp -x \amp 2\!\left(y-x\right) \end{array} \right| = 2\vec i + 2\vec j -6 \vec k, \end{equation*}
writing \(S\) for the disk in the plane enclosed by the circle, Stokesâ€™ Theorem gives
\begin{equation*} \int_C\vec F\cdot d\vec r = \int_S\mbox{curl }\vec F\cdot d\vec A = \int_S(2\vec i + 2\vec j -6 \vec k)\cdot d\vec A. \end{equation*}
Now \(d\vec A = \vec n\,dA\text{,}\) where \(\vec n\) is the unit vector perpendicular to the plane and \(dA\) an area element on the plane, so \(\vec n = \frac1{\sqrt3}(\vec i + \vec j + \vec k)\text{,}\) and
\begin{equation*} \int_C\vec F\,d\vec r = \int_S(2\vec i+2\vec j -6\vec k)\cdot \frac{\vec i+\vec j+\vec k}{\sqrt{3}}\,dA = \int_S \frac{-2}{\sqrt 3}\,dA = \frac{-2}{\sqrt 3}\mbox{ Area of circle} = \frac{-18\pi}{\sqrt{3}}. \end{equation*}
