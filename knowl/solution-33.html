Solution 10.4.5.4.1.

(a),(b) The two tables of values are shown below
\(x =\) -1.1 -1 -0.9
\(y = 2.4\) 0.2248 0.2485 0.2746
\(y = 2.5\) 0.1992 0.2202 0.2433
\(y = 2.6\) 0.1716 0.1896 0.2096
\(x =\) -1.01 -1 -0.99
\(y = 2.49\) 0.2209 0.2231 0.2253
\(y = 2.5\) 0.218 0.2202 0.2224
\(y = 2.51\) 0.215 0.2172 0.2194
Both tables are nearly linear. To check this, observe that the increments in each row (column) are equal, or nearly so. The second table is more linear due to finer data. (c) From either table we see that \(f(-1,2.5)\approx 0.2202\text{.}\) Also, using the second table,
\begin{equation*} f_x(-1,2.5) \approx \frac{f(-0.99,2.5) - f(-1.01,2.5)}{.02} = \frac{0.2224 - 0.218}{0.02} = 0.22, \end{equation*}
\begin{equation*} f_y(-1,2.5) \approx \frac{f(-1,2.51) - f(-1,2.49)}{.02} = \frac{0.2172 - 0.2231}{0.02} = -0.295. \end{equation*}
Using these estimates for the partial derivatives, we get that the local linearization of \(f\) around \((-1,2.5)\) is
\begin{equation*} f(x,y) \approx f(-1,2.5)+f_x(-1,2.5)(x + 1)+f_y(-1,2.5)(y-2.5) = 0.2202 + (0.22)(x + 1) + (-0.295)(y - 2.5). \end{equation*}
Now we use \(f_x = e^{x}\sin\!\left(y\right)\) and \(f_y = e^{x}\cos\!\left(y\right)\text{,}\) giving
\begin{equation*} f_x(-1,2.5) = 0.220166, f_y(-1,2.5) = -0.294724, \end{equation*}
we have the local linearization of \(f\) around \((-1,2.5)\)
\begin{equation*} f(x,y) \approx 0.220166+0.220166\!\left(x-\left(-1\right)\right)+\left(-0.294724\right)\!\left(y-2.5\right). \end{equation*}
Note that these agree to a reasonable number of decimal places.
