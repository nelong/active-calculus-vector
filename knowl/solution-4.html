Solution 9.1.8.13.1.

(a) If \(y\) is constant, we get \(z = 6 y^2 + a\text{,}\) for some value of \(a\text{,}\) so that the cross-sections are upward-opening parabolas. (b) If \(x\) is constant, we get \(z = -2 x^2 + a\text{,}\) for some value of \(a\text{,}\) so that the cross-sections are downward-opening parabolas. (c) If \(z\) is constant and equal to 5, we get \(y = \pm \sqrt{3} x\text{,}\) which are two intersecting straight lines.
