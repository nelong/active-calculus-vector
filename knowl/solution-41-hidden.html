\(f\) is continuous in the region \(R\text{,}\) but \(R\) is not closed and bounded so a special analysis is required. Notice that \(f(x,y)\) tends to \(\infty\) as \((x,y)\) tends farther and farther from the origin or tends toward any point on the \(x\) or \(y\) axis. This suggests that a minimum for \(f\text{,}\) if it exists, can not be too far from the origin or too close to the axes. For example, if \(x > 10\) then \(f(x,y) > 4 x > 40\text{,}\) and if \(y > 10\) then \(f(x,y) > 5 y > 50\text{.}\) If \(0\lt x\lt 0.1\) then \(f(x,y) > 2/x > 20\text{,}\) and if \(0\lt y\lt 0.1\) then \(f(x,y) > 3/y > 30\text{.}\) Since \(f(1,1)=14\text{,}\) a global minimum for \(f\) if it exists must be in the smaller region \(R^\prime: 0.1 \le x \le 10\text{,}\) \(0.1 \le y \le 10\text{.}\) The region \(R^\prime\) is closed and bounded and so have a minimum value at some point in \(R^\prime\text{,}\) and since that value is at most 14, it is also a global minimum for all of \(R\text{.}\) Since the region \(R\) has no boundary, the minimum value must occur at a critical point of \(f\text{.}\) At a critical point we have
\begin{equation*} f_x = -{2\over x^2} + 4 = 0 = f_y = -{3\over y^2} + 5 = 0. \end{equation*}
The only critical point is \((\sqrt{\frac{1}{2}}, \sqrt{\frac{3}{5}})\text{,}\) at which \(f\) achieves the minimum value
\begin{equation*} f(\sqrt{\frac{1}{2}},\sqrt{\frac{3}{5}}) = 2\sqrt{8} + 2\sqrt{15} \end{equation*}
