Solution 10.7.5.8.1.

\(x=2\text{,}\) \(y=0\text{,}\)
\begin{equation*} f_x = 2x + A = 4 + A = 0, \end{equation*}
and
\begin{equation*} f_y = 2y = 0, \end{equation*}
so that \(A = -4\text{.}\) Then, for this to be a minimum, we must have \(D > 0\) and \(f_{xx} > 0\text{.}\) Here, \(f_{xx} = 2\text{,}\) \(f_{yy} = 2\text{,}\) and \(f_{xy} = 0\text{,}\) so
\begin{equation*} D = 4 - 0 = 4 > 0 \end{equation*}
and \(f_{xx} > 0\text{.}\) Thus this is a minimum, as advertised. For it to occur with a \(z\)-coordinate of 45, we must have
\begin{equation*} f(2,0) = -4 + B = 45, \end{equation*}
so that \(B = 49\text{.}\)
