Solution 10.7.5.10.1.

\(f_x = f_y = 0\text{,}\) so each of the three points is a critical point. Then we have \(D = f_{xx} f_{yy} - (f_{xy})^2\text{,}\) so we know that (a) At \((1,3)\text{,}\) \(D \lt 0\text{,}\) so (1,3) is a saddle point. (b) At \((2,1)\text{,}\) \(D \lt 0\text{,}\) so (2,1) is a saddle point. (c) At \((5,7)\text{,}\) \(D \lt 0\text{,}\) so (5,7) is a saddle point.
