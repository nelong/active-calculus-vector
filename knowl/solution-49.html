Solution 10.8.3.1.1.

\(f(x,y)= 4 x - y\) and our equation of constraint is \(g(x,y) = x^2 + 2 y^2 = 66\text{.}\) To optimize \(f(x,y)\) with Lagrange multipliers we solve the following system of equations
\begin{equation*} f_{x}(x,y) = \lambda g_{x}(x,y),\qquad\mbox{so } 4 = 2\lambda x, \end{equation*}
\begin{equation*} f_{y}(x,y) = \lambda g_{y}(x,y),\qquad\mbox{so } -1 = 4\lambda y, \end{equation*}
and
\begin{equation*} g(x,y) = 66,\qquad\mbox{so } x^2 + 2 y^2 = 66. \end{equation*}
Solving for \(\lambda\) gives us
\begin{equation*} \lambda = \frac{2}{ x} = \frac{-1}{4 y}, \end{equation*}
which we can use to find \(x\) in terms of \(y\text{:}\)
\begin{equation*} x = -8\,y. \end{equation*}
Using this relation in our equation of constraint, we can solve for \(y\text{:}\)
\begin{equation*} x^2 + 2 y^2 = 66, \end{equation*}
so
\begin{equation*} (-8\, y)^2 + 2 y^2 = 66, \end{equation*}
and thus
\begin{equation*} ((8)^2 + 2)\, y^2 = 66. \end{equation*}
So, solving for \(y\text{,}\) we have \(y = \pm 1\text{,}\) and \(x = \mp 8\text{.}\) Thus, the critical points are \((-8,1)\) and \((8,-1)\text{.}\) Since the constraint is closed and bounded, maximum and minimum values of \(f\) subject to the constraint exist. Evaluating \(f\) at the critical points, we find that the maximum value is \(f(8,-1) = 4(8) + (1) = 33\) and the minimum value is \(f(-8,1) = -4(8) - (1) = -33\text{.}\)
