Solution 10.8.3.5.1.

\((x,y,z)\) be a point on the paraboloid. The square of the distance from \((x,y,z)\) to the point \((1,2,13)\) is given by
\begin{equation*} f(x,y,z) = (x-1)^2 + (y-2)^2 + (z-13)^2, \end{equation*}
and so we wish to minimize \(f(x,y,z)\) subject to the constraint
\begin{equation*} g(x,y,z) = x^2+y^2-z = 0. \end{equation*}
We look for solutions to the equations \(\nabla f = \lambda\nabla g\) and \(g=0\text{:}\)
\begin{equation*} 2(x-1) = 2\lambda x, \end{equation*}
\begin{equation*} 2(y-2) = 2\lambda y, \end{equation*}
\begin{equation*} 2(z-13) = -\lambda, \end{equation*}
and
\begin{equation*} x^2+y^2-z = 0. \end{equation*}
If \(\lambda=0\text{,}\) the first three equations would imply that \((x,y,z)=(1,2,13)\text{,}\) which does not satisfy the fourth equation and so \(\lambda\ne 0\text{.}\) If collect terms in \(x\) and \(y\) in the first and second equations, respectively, we get
\begin{equation*} (1 - \lambda) x = 1, \end{equation*}
and
\begin{equation*} (1 - \lambda) y = 2. \end{equation*}
Note that we cannot have \(\lambda = 1\text{,}\) \(x = 0\) or \(y = 0\text{,}\) and so can divide these two equation to solve for \(y\text{,}\) getting
\begin{equation*} y = 2 x\text{.} \end{equation*}
If we divide the first and third equations we got from the Lagrange multipliers method, we have
\begin{equation*} \frac{x - 1}{z - 13} = -2x, \end{equation*}
so that
\begin{equation*} z = 13 - \frac12(1 - \frac{1}{x}). \end{equation*}
We can then plug all of these into the last equation, the constraint. This gives
\begin{equation*} x^2 + 4 x^2 - 13 + \frac12(1 - \frac{1}{x}) = 0. \end{equation*}
Multiplying by \(x\) and collecting terms, we have
\begin{equation*} 10 x^3 - 25 x - 1 = 0. \end{equation*}
Let \(h(x) = 10 x^3 - 25 x - 1\text{.}\) Clearly \(h(-3) \lt 0\text{,}\) \(h(-\frac12) > 0\text{,}\) \(h(0) \lt 0\) and \(h(3) > 0\text{.}\) Thus there must be roots lying between \(x = -3\) and \(x = -0.5\text{,}\) between \(x = -0.5\) and \(x = 0\text{,}\) and between \(x = 0\) and \(x = 3\text{.}\) We can locate these numerically using a calculator or computer program, to find
\begin{equation*} x = -1.560746, \quad x = -0.04002564928, \quad\mbox{and}\quad x = 1.600771649. \end{equation*}
Then, using the equations for \(y\) and \(z\) in terms of \(x\text{,}\) we have the corresponding values of \(y\) and \(z\text{,}\) and therefore have three solutions to the Lagrange multiplier equations:
\begin{equation*} x_1 = -1.560746, y_1 = -3.121492, z_1 = 12.17964038258, \end{equation*}
\begin{equation*} x_2 = -0.04002564928, y_2 = -0.08005129856, z_2 = 0.00801026300142782, \end{equation*}
and
\begin{equation*} x_3 = 1.600771649, y_3 = 3.201543298, z_3 = 12.8123493612109. \end{equation*}
The distances (squared) from the paraboloid are
\begin{equation*} d_1 = d(x_1,y_1,z_1) = 33.4600902844735, \end{equation*}
\begin{equation*} d_2 = d(x_2,y_2,z_2) = 174.200064082078, \end{equation*}
and
\begin{equation*} d_3 = d(x_3,y_3,z_3) = 1.83984563344885, \end{equation*}
so that the minimum distance is \(\sqrt{d_3} = 1.35641\text{.}\) Note that the geometric interpretation of this is that it is the length of a perpendicular line segment drawn from the paraboloid to the point.
