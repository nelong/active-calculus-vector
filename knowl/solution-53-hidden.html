(a) Let \(f(x_1,x_2,x_3)=\sum_{i=1}^3{x_i}^2 = {x_1}^2 + {x_2}^2 + {x_3}^2\) and \(g(x_1,x_2,x_3)=\sum_{i=1}^3 x_i = 5\text{.}\) Then \(\nabla f = \lambda\nabla g\) gives
\begin{equation*} 2x_1 = \lambda = 2x_2 = 2x_3. \end{equation*}
so
\begin{equation*} x_1 = x_2 = x_3 = \frac{\lambda}2. \end{equation*}
Then \(x_1+x_2+x_3 = 5\) gives
\begin{equation*} 3\frac{\lambda}{2} = 5\quad\hbox{ so } \quad \lambda = \frac{10}{3}. \end{equation*}
Thus \(x_1 = x_2 = x_3 = \frac{5}{3}\text{.}\) These values of \(x_1, x_2,x_3\) give the minimum (rather than maximum) because the value of without bound as \(x_2,x_2,x_3\to\infty\text{.}\) (b) A similar argument shows that \(\sum_{i=1}^nx_i\) has its minimum value subject to \(\sum_{i=1}^nx_i = 5\) when
\begin{equation*} x_1=x_2=\cdots= x_n=\frac{5}n. \end{equation*}
Then the minimum value is
\begin{equation*} \sum_1^n \left(\frac{5}{n}\right)^2 = n\cdot \left(\frac{5}{n}\right)^2 = \frac{25}{n}. \end{equation*}
