We want to find a lower bound and an upper bound for the volume above each subrectangle. The lower bound for the volume of \(R_{(a,b)}\) is
\begin{equation*} 0.375 ( \mbox{Min of } f \mbox{ on } R_{(a,b)}) \end{equation*}
because the area of \(R_{(a,b)}\) is \(0.5\cdot 0.75=0.375\text{.}\) The function \(f(x,y) = 4+xy\) increases with both \(x\) and \(y\) over the whole region \(R\text{.}\) Thus,
\begin{equation*} \mbox{Min of } f \mbox{ on } R_{(a,b)} = f(a,b) = 4+ab, \end{equation*}
because the minimum on each subrectangle is at the corner closest to the origin. Similarly,
\begin{equation*} \mbox{Max of } f \mbox{ on } R_{(a,b)} = f(a+0.5,b+0.75) = 4 + (a+0.5)(b+0.75). \end{equation*}
So we have
\begin{equation*} \mbox{Lower sum } = \sum_{(a,b)} 0.375(4+ab) = 0.375 \sum_{(a,b)} 4+ab = 24 + 0.375\sum_{(a,b)} ab \end{equation*}
Since \(a=0, 0.5, 1, 1.5\) and \(b= 0, 0.75, 1.5, 2.25\text{,}\) expanding this sum gives
