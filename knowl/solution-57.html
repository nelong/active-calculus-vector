Solution 11.1.5.8.1.

\(\Delta A=1\text{.}\) We shall begin our sum as an upper estimate starting with the lower left corner of the room and continue across the bottom and moving upward using the highest temperature, \(T_i\text{,}\) in each case. So the upper Riemann sum becomes
\begin{equation*} \sum_{i=1}^{25} T_i \Delta A = T_1 \Delta A + T_2 \Delta A + T_3 \Delta A + \cdots + T_{25} \Delta A \end{equation*}
\begin{equation*} = \Delta A (T_1 + T_2 + T_3 + \cdots + T_{25}). \end{equation*}
Thus, taking the maximum value for each region (and, where the temperatures are not bounded in the box, taking the largest value plus one), we have
\begin{equation*} \mbox{sum} = (1)(30 + 28 + 27 + 26 + 26 + 28 + 27 + 26 + 26 + 25 + 26 + 26 + 25 + 25 + 25 \end{equation*}
\begin{equation*} + 25 + 24 + 24 + 23 + 23 + 24 + 23 + 22 + 22 + 23) = 629. \end{equation*}
In the same way, the lower Riemann sum is formed by taking the lowest temperature (and, where the temperatures are not bounded in the box, taking the largest value minus one), \(t_i\text{,}\) in each case:
\begin{equation*} \sum_{i=1}^{25} t_i \Delta A = t_1 \Delta A + t_2 \Delta A + t_3 \Delta A + \cdots + t_{25} \Delta A \end{equation*}
\begin{equation*} = \Delta A (t_1 + t_2 + t_3 + \cdots + t_{25}). \end{equation*}
Thus, taking the minimum value for each region, we have
\begin{equation*} \mbox{sum} = (1)(27 + 26 + 26 + 25 + 25 + 26 + 25 + 25 + 25 + 24 + 24 + 24 + 23 + 23 + 23 \end{equation*}
\begin{equation*} + 23 + 22 + 22 + 22 + 22 + 21 + 20 + 19 + 20 + 21) = 583. \end{equation*}
So, averaging the upper and lower sums we get: \(606\text{.}\) To compute the average temperature, we divide by the area of the room,
\begin{equation*} \mbox{Average temperature} = \frac{606}{25} = 24.24. \end{equation*}
That is, approximately \(24.24^{\circ}\)C.
