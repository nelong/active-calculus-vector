Then we can calculate the lower and upper sum by taking the smallest and largest values on each rectangle:
\begin{equation*} \mbox{Riemann sum} = \sum f(x_i,y_i)\Delta x\Delta y, \end{equation*}
so the lower sum is
\begin{equation*} \mbox{Lower} = (5 + 7 + 3 + 5)(0.2)(0.4) = (20)(0.2)(0.4) = 1.6, \end{equation*}
and the upper sum is
\begin{equation*} \mbox{Upper} = (9 + 12 + 7 + 18)(0.2)(0.4) = (46)(0.2)(0.4) = 3.68. \end{equation*}
