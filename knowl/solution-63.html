Solution 11.4.6.10.1.

(a) The probability is
\begin{equation*} \int_0^1\int_{1/3}^1 {2 \over 3} (x + 2y) \,dx\,dy = \int_0^1 {2\over3} ({1\over2}x^2 + 2xy)\bigr|^1_{1/3}\,dy \end{equation*}
\begin{equation*} = \int_0^1 {2\over3} \left[({1\over2} + 2y) - ({1\over 18} + {\frac{2}{3}}y)\right]\,dy \end{equation*}
\begin{equation*} = {2\over3}\int_0^1 (\frac{8}{18} + \frac{4}{3} y) \, dy = {2\over3} \left(\frac{8}{18} y + \frac{2}{3} y^2\right)\bigr|_0^1 \end{equation*}
\begin{equation*} = {2\over3} \left(\frac{10}{9}\right) = \frac{20}{27}. \end{equation*}
(b) It is easier to calculate the probability that \(x \lt \frac1{3} + y\) does not happen, that is, the probability that \(x \ge \frac1{3} + y\text{,}\) and subtract it from 1. The probability that \(x \ge \frac1{3} + y\) is
\begin{equation*} \int_{0}^{2/3}\int_{y+1/3}^{1} {2\over3} (x+2y) \,dx\,dy = \int_{0}^{2/3} (\frac13\,x^2 + \frac43\,xy) \bigr|_{y+(1/3)}^{1} \,dy \end{equation*}
\begin{equation*} = \int_{0}^{2/3} \frac13\left(1 - (\frac1{3} + y)^2\right) + \frac43(y)(1 - (\frac1{3} + y)) \,dy \end{equation*}
\begin{equation*} = \int_{0}^{2/3} -\frac53\,y^2 + \frac{2}{3} y + \frac{8}{27} \, dy \end{equation*}
\begin{equation*} = (-\frac59\,y^3 + \frac{1}{3}\, y^2 + \frac{8}{27}\,y)\bigr|_{0}^{2/3} \end{equation*}
\begin{equation*} = \frac{44}{243}. \end{equation*}
Thus, the probability that \(x \lt \frac1{3}+y\) is \(1-(\frac{44}{243}) = \frac{199}{243}\text{.}\)
