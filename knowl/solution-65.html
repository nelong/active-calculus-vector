Solution 11.5.4.10.1.

(a) Since this is a partially-circular region, we use polar coordinates. This gives
\begin{equation*} \int_{\pi/3}^{\pi/2}\int_{0}^{1} f(r\cos(\theta),r\sin(\theta))\,r\,dr\,d\theta. \end{equation*}
(b) Since this is a rectangular region, we use Cartesian coordinates. This gives
\begin{equation*} \int_{1}^{5}\int_{4}^{5} f(x,y)\,dy\,dx. \end{equation*}
