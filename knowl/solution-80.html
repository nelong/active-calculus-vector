Solution 11.9.5.2.1.

\begin{equation*} \left|\frac{\partial (x,y,z)}{\partial (s,t,u)}\right| = \left|\begin{array}{ccc} x_s \amp x_t \amp x_u\\ y_s \amp y_t \amp y_u\\ z_s \amp z_t \amp z_u \end{array}\right| =\left|\begin{array}{ccc} -5 \amp 4 \amp 3\\ -5 \amp -1 \amp 1\\ -4 \amp 5 \amp -4 \end{array}\right|. \end{equation*}
This \(3 \times 3\) determinant is computed the same way as for the cross product, with the entries \(-5,4,3\) in the first row playing the same role as \(\vec i,\vec j,\vec k\text{.}\) We get
\begin{equation*} \frac{\partial (x,y,z)}{\partial (s,t,u)}= ((-1)(-4)-(1)(5))-5 + ((1)(-4)-(-5)(-4))4 + ((-5)(5)-(-1)(-4))3 = -178. \end{equation*}
