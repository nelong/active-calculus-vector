Solution 11.9.5.7.1.

\(T\) is defined by the inequalities
\begin{equation*} 0 \le s = ax \le 1 \qquad 0\le t=by\le 1 \end{equation*}
that correspond to the inequalities
\begin{equation*} 0\le x \le 1/a = 45 \qquad 0\le y \le 1/b = 20 \end{equation*}
that define \(R\text{.}\) Thus \(a=1/45\) and \(b=1/20\text{.}\) Then, we have \(x = 45 s\) and \(y = 20 t\text{,}\) so
\begin{equation*} \left\vert \frac{\partial(x,y)}{\partial(s,t)}\right\vert = \left|\begin{array}{cc} 45 \amp 0 \\ 0 \amp 20 \end{array}\right| = 900. \end{equation*}
