Activity 11.4.2.

Find all second order partial derivatives of the following functions. For each partial derivative you calculate, state explicitly which variable is being held constant.
  1. \(\displaystyle f(x,y) = x^2y^3\)
  2. \(\displaystyle f(x,y) = y\cos(x)\)
  3. \(\displaystyle g(s,t) = st^3 + s^4\)
  4. How many second order partial derivatives does the function \(h\) defined by \(h(x,y,z) = 9x^9z-xyz^9 + 9\) have? Find \(h_{xz}\) and \(h_{zx}\) (you do not need to find the other second order partial derivatives).
Solution.
  1. To begin, we calculate the first order partials, holding \(y\) constant to calculate \(f_x\) and \(x\) constant to calculate \(f_y\text{:}\)
    \begin{align*} f_x(x,y) \amp = 2xy^3\\ f_y(x,y) \amp = 3x^2y^2\text{.} \end{align*}
    The second order partial derivatives are partial derivatives of the first order partial derivatives:
    \begin{align*} f_{xx}(x,y) \amp = (f_x)_x(x,y) = 2y^3\\ f_{xy}(x,y) \amp = (f_x)_y(x,y) = 6xy^2\\ f_{yx}(x,y) \amp = (f_y)_x(x,y) = 6xy^2\\ f_{yy}(x,y) \amp = (f_y)_y(x,y) = 6x^2y\text{.} \end{align*}
  2. To begin, we calculate the first order partials:
    \begin{align*} f_x(x,y) \amp = -y\sin(x)\\ f_y(x,y) \amp = \cos(x)\text{.} \end{align*}
    The second order partial derivatives are partial derivatives of the first order partial derivatives:
    \begin{align*} f_{xx}(x,y) \amp = (f_x)_x(x,y) = -y\cos(x)\\ f_{xy}(x,y) \amp = (f_x)_y(x,y) = -\sin(x)\\ f_{yx}(x,y) \amp = (f_y)_x(x,y) = -\sin(x)\\ f_{yy}(x,y) \amp = (f_y)_y(x,y) = 0\text{.} \end{align*}
  3. To begin, we calculate the first order partials, holding \(t\) constant to calculate \(g_s\) and \(s\) constant to calculate \(g_t\text{:}\)
    \begin{align*} g_s(s,t) \amp = t^3+4s^3\\ g_t(s,t) \amp = 3st^2\text{.} \end{align*}
    The second order partial derivatives are partial derivatives of the first order partial derivatives:
    \begin{align*} g_{ss}(s,t) \amp = (g_s)_s(s,t) = 12s^2\\ g_{st}(s,t) \amp = (g_s)_t(s,t) = 3t^2\\ g_{ts}(s,t) \amp = (g_t)_s(s,t) = 3t^2\\ g_{tt}(s,t) \amp = (g_t)_t(s,t) = 6st\text{.} \end{align*}
  4. Since \(h\) is a function of three independent variables, \(h\) has three first-order partial derivatives: \(h_x, h_y\text{,}\) and \(h_z\text{.}\) Each of those has three partial derivatives, so \(h\) has nine second-order partial derivatives.
    To find \(h_{xz}\text{,}\) we start by calculating \(h_x\text{,}\) holding \(y, z\) constant and differentiating with respect to \(x\text{:}\)
    \begin{equation*} h_x (x, y, z) = 81x^8z - yz^9\text{.} \end{equation*}
    We then take the partial derivative of \(h_x\) with respect to \(z\text{,}\) holding \(x\) and \(y\) constant:
    \begin{equation*} h_{xz} (x, y, z) = 81x^8 - 9yz^8\text{.} \end{equation*}
    Similarly, to find \(h_{zx}\text{,}\) we start by calculating \(h_z\text{,}\) holding \(x, y\) constant and differentiating with respect to \(z\text{:}\)
    \begin{equation*} h_z (x, y, z) = 9x^9 - 9xyz^8\text{.} \end{equation*}
    We then take the partial derivative of \(h_z\) with respect to \(x\text{,}\) holding \(y\) and \(z\) constant:
    \begin{equation*} h_{zx} (x, y, z) = 81x^8 - 9yz^8\text{.} \end{equation*}
