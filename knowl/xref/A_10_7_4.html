Activity 11.8.3.

Recall that the Second Derivative Test for single-variable functions states that if \(x_0\) is a critical point of a function \(f\) so that \(f'(x_0)=0\) and if \(f''(x_0)\) exists, then
  • if \(f''(x_0) \lt 0\text{,}\) \(x_0\) is a local maximum,
  • if \(f''(x_0) > 0\text{,}\) \(x_0\) is a local minimum, and
  • if \(f''(x_0) = 0\text{,}\) this test yields no information.
Our goal in this activity is to understand a similar test for classifying extreme values of functions of two variables. Consider the following three functions:
\begin{equation*} f_1(x,y) = 4-x^2-y^2, \ \ f_2(x,y) = x^2+y^2, \ \ f_3(x,y) = x^2-y^2. \end{equation*}
You can verify that each function has a critical point at the origin \((0,0)\text{.}\) You should check this.
  1. The graphs of these three functions are shown in FigureÂ 11.8.6, with \(z=4-x^2-y^2\) at left, \(z=x^2+y^2\) in the middle, and \(z=x^2-y^2\) at right. Use the graphs to decide if a function has a local maximum, local minimum, saddle point, or none of the above at the origin.
  2. There is no single second derivative of a function of two variables, so we consider a quantity that combines the second order partial derivatives. Let \(D = f_{xx}f_{yy} - f_{xy}^2\text{.}\) Calculate \(D\) at the origin for each of the functions \(f_1\text{,}\) \(f_2\text{,}\) and \(f_3\text{.}\) What difference do you notice between the values of \(D\) when a function has a maximum or minimum value at the origin versus when a function has a saddle point at the origin?
  3. Now consider the cases where \(D \gt 0\text{.}\) It is in these cases that a function has a local maximum or minimum at a point. What is necessary in these cases is to find a condition that will distinguish between a maximum and a minimum. In the cases where \(D \gt 0\) at the origin, evaluate \(f_{xx}(0,0)\text{.}\) What value does \(f_{xx}(0,0)\) have when \(f\) has a local maximum value at the origin? When \(f\) has a local minimum value at the origin? Explain why. (Hint: This should look very similar to the Second Derivative Test for functions of a single variable.) What would happen if we considered the values of \(f_{yy}(0,0)\) instead?
Solution.
  1. From the graphs, we can conclude that \(z=4-x^2-y^2\) has a local maximum at the origin, \(z=x^2+y^2\) has a local minimum at the origin, and \(z=x^2-y^2\) has neither a maximum nor a minimum at the origin.
  2. For \(f(x, y) = 4-x^2-y^2\text{,}\) we have
    \begin{equation*} f_{xx}(0,0) = -2, f_{yy}(0,0) = -2,\ \mbox{and} \ f_{xy}(0,0) = 0\text{,} \end{equation*}
    giving \(D = (-2)(-2) - 0^2 = 4 \gt 0\text{.}\)
    For \(f(x, y) = x^2+y^2\text{,}\) we have
    \begin{equation*} f_{xx}(0,0) = 2, f_{yy}(0,0) = 2,\ \mbox{and} \ f_{xy}(0,0) = 0\text{,} \end{equation*}
    giving \(D = 2\cdot2 - 0^2 = 4 \gt 0\text{.}\)
    For \(f(x, y) = x^2 - y^2\text{,}\) we find that
    \begin{equation*} f_{xx}(0,0) = 2, f_{yy}(0,0) = -2,\ \mbox{and} \ f_{xy}(0,0) = 0\text{,} \end{equation*}
    giving \(D = 2\cdot(-2) - 0^2 = -4 \lt 0\text{.}\)
    Combining the above information with our conclusions from the previous part, we see that \(D \gt 0\) at a point where the function has a maximum or minimum value but \(D \lt 0\) at a point where the function has a saddle point.
  3. For our first function, \(f(x, y) = 4-x^2-y^2\text{,}\) \(D \gt 0\) and \(f_{xx}(0,0) = -2 \lt 0\text{,}\) and \(f\) has a local maximum at \((0, 0)\text{.}\) For our second function, \(f(x, y) = x^2+y^2\text{,}\) \(D \gt 0\) and \(f_{xx}(0,0) = 2 \gt 0\text{,}\) and \(f\) has a local minimum at \((0, 0)\text{.}\)
    To understand the connection, recall that \(f_{x}(0, 0) \) is the slope of the trace of \(f(x,y)\) in the plane \(y = 0\text{.}\) In turn, \(f_{xx}\) gives us information about the concavity of that trace. If \(f_{xx}(0,0) \lt 0\text{,}\) we might expect that \(f\) appears to be concave down in the \(x\)-direction, which, like the single variable case, may indicate a local maximum at \((0, 0)\text{.}\) On the other hand, if \(f_{xx}(0,0) \gt 0\text{,}\) we might expect that \(f\) appears to be concave up in the \(x\)-direction, which like the single variable case, may indicate a local minimum at \((0, 0)\text{.}\) Since we have a function of two variables, we need to consider the other direction as well. Note that in both of our examples above, the sign of \(f_{xx}(0,0)\) is the same as the sign of \(f_{yy}(0, 0)\text{.}\) In fact, we need these two partials to have the same sign to get \(D \gt 0\text{.}\)
