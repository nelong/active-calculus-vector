Activity 11.8.4.

Find the critical points of the following functions and use the Second Derivative Test to classify the critical points.
  1. \(\displaystyle f(x,y) = 3x^3+y^2-9x+4y\)
  2. \(\displaystyle f(x,y) = xy + \frac{2}{x} + \frac{4}{y}\)
  3. \(f(x,y) = x^3 + y^3 - 3xy\text{.}\)
Solution.
  1. To find the critical points, we determine the points in the domain of \(f\) where \(\nabla f = \vzero\) or \(\nabla f\) does not exist. Now
    \begin{equation*} \nabla f = \langle 9x^2-9, 2y+4 \rangle\text{,} \end{equation*}
    and so \(\nabla f = \vzero\) when \(9x^2-9 = 0\) and \(2y+4 = 0\text{.}\) The critical points of \(f\) are then \((1,-2)\) and \((-1,-2)\text{.}\) To classify these critical points with the Second Derivative Test, we need the second order partial derivatives of \(f\text{.}\) Differentiating the first order partial derivatives gives us
    \begin{equation*} \frac{\partial^2 f}{\partial x^2} = 18x, \ \ \ \frac{\partial^2 f}{\partial y^2} = 2, \ \ \text{ and } \ \ \frac{\partial^2 f}{\partial x \partial y} = 0\text{.} \end{equation*}
    Then
    \begin{equation*} D(1,-2) = f_{xx}(1,-2)f_{yy}(1,-2) - \left(f_{xy}(1,-2)\right)^2 = (18)(2) - (0)^2 = 36\text{.} \end{equation*}
    That \(D(1,-2) \gt 0\) and \(f_{xx}(1,-2) \gt 0\) tells us that \(f\) has a relative minimum at \((1,-2)\text{.}\) Also,
    \begin{equation*} D(-1,-2) = f_{xx}(-1,-2)f_{yy}(-1,-2) - \left(f_{xy}(-1,-2)\right)^2 = (-18)(2) - (0)^2 = -36\text{.} \end{equation*}
    That \(D(-1,-2) \lt 0\) tells us that \(f\) has a saddle point at \((-1,-2)\text{.}\)
  2. To find the critical points, we determine the points in the domain of \(f\) (\(x \neq 0\) and \(y \neq 0\)) where \(\nabla f = \vzero\) or \(\nabla f\) does not exist. Now
    \begin{equation*} \nabla f = \left\langle y-\frac{2}{x^2}, x-\frac{4}{y^2} \right\rangle\text{,} \end{equation*}
    and so \(\nabla f = \vzero\) when
    \begin{align*} y-\frac{2}{x^2} \amp = 0\\ x-\frac{4}{y^2} \amp = 0\text{.} \end{align*}
    The first equation tells us that \(y = \frac{2}{x^2}\text{.}\) Substituting into the second equation shows that
    \begin{align*} x - \frac{4}{\left(\frac{2}{x^2}\right)^2} \amp = 0\\ x - \frac{4x^4}{4} \amp = 0\\ x - x^4 \amp = 0\\ x(1-x^3) \amp = 0\text{.} \end{align*}
    So \(x = 0\) or \(1-x^3 = 1\text{.}\) This gives us \(x=0\) or \(x=1\text{.}\) Now \(x=0\) is not in the domain of \(f\text{,}\) so we only consider \(x=1\text{.}\) Then \(y = \frac{2}{1^2} = 2\text{.}\) Thus, the only critical point of \(f\) is \((1,2)\text{.}\) To classify this critical point with the Second Derivative Test, we need the second order partial derivatives of \(f\text{.}\) Differentiating the first order partial derivatives gives us
    \begin{equation*} \frac{\partial^2 f}{\partial x^2} = \frac{4}{x^3}, \ \ \ \frac{\partial^2 f}{\partial y^2} = \frac{8}{y^3}, \ \ \text{ and } \ \ \frac{\partial^2 f}{\partial x \partial y} = 1\text{.} \end{equation*}
    Then
    \begin{equation*} D(1,2) = f_{xx}(1,2)f_{yy}(1,2) - \left(f_{xy}(1,2)\right)^2 = (4)(1) - (1)^2 = 3\text{.} \end{equation*}
    That \(D(1,2) \gt 0\) and \(f_{xx}(1,2) \gt 0\) tells us that \(f\) has a relative minimum at \((1,2)\text{.}\)
  3. To find the critical points, we determine the points in the domain of \(f\) where \(\nabla f = \vzero\) or \(\nabla f\) does not exist. Now
    \begin{equation*} \nabla f = \langle 3x^2-3y, 3y^2 - 3x \rangle\text{,} \end{equation*}
    and so \(\nabla f = \vzero\) when \(3x^2-3y = 0\) and \(3y^2 - 3x = 0\text{.}\) Solving these equations simultaneously yields the critical points of \((0, 0)\) and \((1, 1)\text{.}\) To classify these critical points with the Second Derivative Test, we need the second order partial derivatives of \(f\text{.}\) Differentiating the first order partial derivatives gives us
    \begin{equation*} \frac{\partial^2 f}{\partial x^2} = 6x, \ \ \ \frac{\partial^2 f}{\partial y^2} = 6y, \ \ \text{ and } \ \ \frac{\partial^2 f}{\partial x \partial y} = -3\text{.} \end{equation*}
    Then
    \begin{equation*} D(0, 0) = f_{xx}(0, 0)f_{yy}(0 0) - \left(f_{xy}(0, 0)\right)^2 = (0)(0) - (-3)^2 = -9\text{.} \end{equation*}
    That \(D(0, 0) \lt 0\) tells us that \(f\) has a saddle point at \((0, 0)\text{.}\) Also,
    \begin{equation*} D(1,1) = f_{xx}(1, 1)f_{yy}(1, 1) - \left(f_{xy}(1, 1)\right)^2 = (6)(6) - (-3)^2 = 36 - 9 = 27\text{.} \end{equation*}
    Since \(D(1, 1) \gt 0\) and \(f_{xx}(1,1) \gt 0\text{,}\) we determine that \(f\) has a local minimum at \((1, 1)\text{.}\)
