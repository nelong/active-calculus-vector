Activity 9.4.3.

Determine each of the following.
  1. The length of the vector \(\vu=\langle 1, 2, -3 \rangle\) using the dot product.
  2. The angle between the vectors \(\vu =\langle 1, 2 \rangle\) and \(\vv = \langle 4, -1 \rangle\) to the nearest tenth of a degree.
  3. The angle between the vectors \(\vy =\langle 1, 2, -3 \rangle\) and \(\vz = \langle -2, 1, 1 \rangle\) to the nearest tenth of a degree.
  4. If the angle between the vectors \(\vu\) and \(\vv\) is a right angle, what does the expression \(\vu\cdot\vv=|\vu||\vv|\cos(\theta)\) say about their dot product?
  5. If the angle between the vectors \(\vu\) and \(\vv\) is acute—that is, less than \(\pi/2\)—what does the expression \(\vu\cdot\vv=|\vu||\vv|\cos(\theta)\) say about their dot product?
  6. If the angle between the vectors \(\vu\) and \(\vv\) is obtuse—that is, greater than \(\pi/2\)—what does the expression \(\vu\cdot\vv=|\vu||\vv|\cos(\theta)\) say about their dot product?
Solution.
  1. \(|\vu|^2 = \vu\cdot\vu\text{.}\)
    So \(|\vu| = \sqrt{\vu\cdot\vu} = \sqrt{\langle 1, 2, -3 \rangle\cdot\langle 1, 2, -3 \rangle} = \sqrt{(1)(2) + (2)(2) + (-3)(-3)} = \sqrt{14}\text{.}\)
  2. \(\vu\cdot\vv =|\vu||\vv|\cos(\theta)\text{.}\)
    \(\vu\cdot\vv = \langle 1, 2 \rangle\cdot\langle 4, -1 \rangle = (1)(4) + (2)(-1) = 2\text{.}\)
    \(|\vu| = \sqrt{(1)^2 + (2)^2} = \sqrt{5}\) and \(|\vv| = \sqrt{(4)^2 + (-1)^2} = \sqrt{17}\text{.}\)
    Then \(2 = \sqrt{5} * \sqrt{17} * \cos(\theta)\text{.}\) So \(\theta = \cos^{-1}\left(\frac{2}{\sqrt{5} * \sqrt{17}}\right) = 77.471^{\circ}\text{.}\)
  3. \(\vy\cdot\vz = \langle 1, 2, -3 \rangle\cdot\langle -2, 1, 1 \rangle = (1)(-2) + (2)(1) + (-3)(1) = -3\text{.}\)
    \(|\vy| = \sqrt{(1)^2 + (2)^2 + (-3)^2} = \sqrt{14}\) and \(|\vz| = \sqrt{(-2)^2 + (1)^2 + (1)^2} = \sqrt{6}\text{.}\)
    Then \(-3 = \sqrt{14} * \sqrt{6} * \cos(\theta)\text{.}\) So \(\theta = \cos^{-1}\left(\frac{-3}{\sqrt{14} * \sqrt{6}}\right) = 109.107^{\circ}\text{.}\)
  4. If the angle between 2 vectors is a right angle, the angle measure is \(\frac{\pi}{2}\text{.}\) \(\cos(\frac{\pi}{2}) = 0\text{.}\) Therefore \(\vu\cdot\vv = |\vu||\vv| * 0 = 0\text{.}\) So \(\vu\cdot\vv = 0\text{.}\)
  5. If the angle between 2 vectors is acute, the angle measure is between \(0\) and \(\frac{\pi}{2}\text{.}\) \(\cos(\theta)\) is between 0 and 1. Therefore \(\vu\cdot\vv = |\vu||\vv| \cos(\theta)\) is positive. So \(\vu\cdot\vv\) is positive.
  6. If the angle between 2 vectors is obtuse, the angle measure is between \(\frac{\pi}{2}\) and \(\pi\text{.}\) \(\cos(\theta)\) is between -1 and 0. Therefore \(\vu\cdot\vv = |\vu||\vv| \cos(\theta)\) is negative. So \(\vu\cdot\vv\) is negative.
