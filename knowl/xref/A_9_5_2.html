Activity 9.6.2.

Let \(P_1 = (1,2,-1)\) and \(P_2 = (-2,1,-2)\text{.}\) Let \(\mathcal{L}\) be the line in \(\R^3\) through \(P_1\) and \(P_2\text{,}\) and note that three snapshots of this line are shown in FigureÂ 9.6.5.
  1. Find a direction vector for the line \(\mathcal{L}\text{.}\)
  2. Find a vector equation of \(\mathcal{L}\) in the form \(\vr(t) = \vr_0 + t\vv\text{.}\)
  3. Consider the vector equation \(\vs(t) = \langle -5, 0, -3 \rangle + t \langle 6, 2, 2 \rangle.\) What is the direction of the line given by \(\vs(t)\text{?}\) Is this new line parallel to line \(\mathcal{L}\text{?}\)
  4. Do \(\vr(t)\) and \(\vs(t)\) represent the same line, \(\mathcal{L}\text{?}\) Explain.
Solution.
  1. The direction vector the for line \(\mathcal{L}\) is \(\langle ((-2) - (1), (1) - (2), (-2) - (-1)) \rangle = \angle -3, -1, -1 \rangle\text{.}\)
  2. A vector equation of \(\mathcal{L}\) is \(\vr(t) = \langle 1, 2, -1 \rangle + t \langle -3, -1, -1 \rangle\text{.}\)
  3. The direction of the line given by \(s(t)\) is given by the vector that is multiplied by \(t\text{,}\) which is \(\langle 6, 2, 2 \rangle\text{.}\)
    This line is parallel to the line \(\mathcal{L}\) because the \(\langle 6, 2, 2 \rangle = -2 * \langle -3, -1, -1 \rangle\) meaning the direction vectors are scalar multiples of one another.
  4. We know that the lines are parallel to each other from part \(c\text{.}\) Now we need to see if they have a point in common, either \((1, 2, -1)\) or \((-2, 1, -2)\text{,}\) by determining if there is a \(t\)-value such that \(\langle 1, 2, -1 \rangle = \vs(t)\) or \(\langle -2, 1, -2 \rangle = \vs(t)\text{.}\)
    \(\langle 1, 2, -1 \rangle = \langle -5, 0, 3 \rangle + t * \langle 6, 2, 2 \rangle\text{.}\)
    Subtract \(\langle -5, 0, 3 \rangle\) from both sides to get \(\langle 6, 2, 2 \rangle = t * \langle 6, 2, 2 \rangle\text{.}\) This equation is true when \(t = 1\text{.}\) Therefore, \(\r(t)\) and \(\vs(t)\) are the same lines.
