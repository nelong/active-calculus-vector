Activity 10.1.2.

The same curve can be represented with different parameterizations. Use appropriate technology to plot each of the curves generated by the following vector-valued functions for values of \(t\) from \(0\) to \(2 \pi \text{.}\) For each example you should write a few sentences to explain how the graphs are alike and how they are different.
  1. \(\displaystyle \vr(t) = \langle \sin(t), \cos(t) \rangle\)
  2. \(\displaystyle \vr(t) = \langle \sin(2t), \cos(2t) \rangle\)
  3. \(\displaystyle \vr(t) = \langle \cos(t+\pi), \sin(t+\pi) \rangle\)
  4. \(\displaystyle \vr(t) = \langle \cos(t^2), \sin(t^2) \rangle\)
