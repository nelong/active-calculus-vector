Activity 10.1.2.

The same curve can be represented with different parameterizations. Use appropriate technology to plot the curves generated by the following vector-valued functions for values of \(t\) from \(0\) to \(2 \pi \text{.}\) Compare and contrast the graphs â€” explain how they are alike and how they are different.
  1. \(\displaystyle \vr(t) = \langle \sin(t), \cos(t) \rangle\)
  2. \(\displaystyle \vr(t) = \langle \sin(2t), \cos(2t) \rangle\)
  3. \(\displaystyle \vr(t) = \langle \cos(t+\pi), \sin(t+\pi) \rangle\)
  4. \(\displaystyle \vr(t) = \langle \cos(t^2), \sin(t^2) \rangle\)
Solution.
  1. This vector-valued function is a circle traced in the clockwise direction with radius \(1\text{,}\) initial point \((0, 1)\text{,}\) and terminal point \((0, 1)\text{.}\)
  2. This vector-valued function is a circle traced in the clockwise direction with radius \(1\text{,}\) initial point \((0, 1)\text{,}\) and terminal point \((0, 1)\text{,}\) but the circle is traced twice since the period of the parametric trigonometric functions are \(\pi\text{.}\)
  3. This vector-valued function is a circle traced in the clockwise direction with radius \(1\text{,}\) initial point \((0, -1)\text{,}\) and terminal point \((0, -1)\text{.}\)
  4. This vector-valued function is a circle traced in the clockwise direction with radius \(1\text{,}\) initial point \((0, 1)\text{,}\) and terminal point \((sin(4(\pi)^2), cos(4(\pi)^2))\text{.}\) The circle is traced multiple times.
