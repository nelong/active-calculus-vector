Activity 10.2.6.

Suppose a moving object in space has its velocity given by
\begin{equation*} \vv(t) = (-2\sin(2t)) \vi + (2 \cos(t)) \vj + \left(1 - \frac{1}{1+t}\right) \vk. \end{equation*}
A graph of the position of the object for times \(t\) in \([-0.5,3]\) is shown in Figure 10.2.6. Suppose further that the object is at the point \((1.5,-1,0)\) at time \(t=0\text{.}\)
  1. Determine \(\va(t)\text{,}\) the acceleration of the object at time \(t\text{.}\)
  2. Determine \(\vr(t)\text{,}\) position of the object at time \(t\text{.}\)
  3. Compute and sketch the position, velocity, and acceleration vectors of the object at time \(t=1\text{,}\) using Figure 10.2.6.
  4. Finally, determine the vector equation for the tangent line, \(\vL(t)\text{,}\) that is tangent to the position curve at \(t = 1\text{.}\)
Solution.
  1. Since acceleration is the derivative of the velocity we have
    \begin{equation*} \va(t) = \vv'(t) = -4\cos(2t)\vi -2\sin(t) \vj + \left(\frac{1}{(1+t)^2}\right) \vk\text{.} \end{equation*}
  2. The position of the object is an antiderivative of the velocity, so
    \begin{equation*} \vr(t) = \int \vv(t) \, dt = \cos(2t) \vi + 2\sin(t) \vj + \left( t - \ln|1+t| \right) \vk + \vC \end{equation*}
    for some constant \(\vC\text{.}\) Since \(\vr(0) = \langle 1.5, -1, 0 \rangle\text{,}\) we have
    \begin{equation*} \langle 1.5, -1, 0 \rangle = \vr(0) = \langle 1, 0, 0 \rangle + \vC \end{equation*}
    and so \(\vC = \langle 0.5, -1, 0 \rangle\text{.}\) Therefore,
    \begin{equation*} \vr(t) = (\cos(2t)+0.5) \vi + (2\sin(t)-1) \vj + \left( t - \ln|1+t| \right) \vk\text{.} \end{equation*}
  3. The position \(\vr(1) = (\cos(2)+1) \vi + (2\sin(1)-1) \vj + (1-ln(2)) \vk\text{,}\) velocity \(\vv(1) = (-2\sin(2)) \vi + (2\cos(1)) \vj + \left(\frac{1}{2}\right) \vk\text{,}\) and acceleration vector \(\va(1) = (-4\cos(2)) \vi - (2\sin(1)) \vj + \left(\frac{1}{4}\right) \vk\) of the object at time \(t=1\) are shown below.
  4. The vector \(\vv(1)\) is a direction vector for \(\vL(t)\) and so
    \begin{equation*} \vL(t) = \vr(1) + t\vv(1) = \langle \cos(2)+\frac{1}{2}, 2\sin(1)-1, 1-\ln(2) \rangle + t \left\langle -2\sin(2), 2\cos(1), + \frac{1}{2} \right\rangle\text{.} \end{equation*}
