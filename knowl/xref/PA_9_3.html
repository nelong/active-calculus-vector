For two-dimensional vectors \(\vu=\langle u_1,u_2 \rangle\) and \(\vv=\langle v_1, v_2 \rangle\text{,}\) the dot product is simply the scalar obtained by
\begin{equation*} \vu\cdot\vv = u_1v_1 + u_2v_2. \end{equation*}
  1. If \(\vu=\langle 3, 4 \rangle\) and \(\vv=\langle -2, 1 \rangle\text{,}\) find the dot product \(\vu\cdot\vv\text{.}\)
  2. Find \(\vi\cdot\vi\) and \(\vi\cdot\vj\text{.}\)
  3. If \(\vu=\langle 3, 4 \rangle\text{,}\) find \(\vu\cdot\vu\text{.}\) How is this related to \(|\vu|\text{?}\)
  4. On the axes in Figure 9.4.1, plot the vectors \(\vu=\langle 1, 3 \rangle\) and \(\vv=\langle -3, 1 \rangle\text{.}\) Then, find \(\vu\cdot\vv\text{.}\) What is the angle between these vectors?
  5. On the axes in Figure 9.4.2, plot the vector \(\vu=\langle 1, 3 \rangle\text{.}\) For each of the following vectors \(\vv\text{,}\) plot the vector on Figure 9.4.2 and then compute the dot product \(\vu\cdot\vv\text{.}\)
    • \(\vv=\langle 3, 2 \rangle\text{.}\)
    • \(\vv=\langle 3, 0 \rangle\text{.}\)
    • \(\vv=\langle 3,-1 \rangle\text{.}\)
    • \(\vv=\langle 3,-2 \rangle\text{.}\)
    • \(\vv=\langle 3,-4 \rangle\text{.}\)
  6. Based upon the previous part of this activity, what do you think is the sign of the dot product in the following three cases shown in Figure 9.4.3?
Solution.
  1. \(\vu\cdot\vv = (3)(-2) + (4)(1) = -2\text{.}\)
  2. \(\vi\cdot\vi = (1)(1) + (0)(0) = 0\text{.}\)
    \(\vi\cdot\vj = (1)(0) + (0)(1) =0\text{.}\)
  3. \(\vu\cdot\vu = (3)(3) + (4)(4) = 25\text{.}\)
    \(|\vu| = \sqrt{(3)^2 + (4)^2} = \sqrt{25} = 5\text{,}\) so \(\vu\cdot\vu = (|\vu|)^2\text{.}\)
  4. \(\vu\cdot\vv = (1)(-3) + (3)(1) = 0\text{.}\) The angle between the vectors is \(90^{\circ}\text{.}\)
  5. \(\vu\cdot\vv = (1)(3) + (3)(2) = 9\text{.}\)
    \(\vu\cdot\vv = (1)(3) + (3)(0)) = 3\text{.}\)
    \(\vu\cdot\vv = (1)(3) + (3)(-1) = 0\text{.}\)
    \(\vu\cdot\vv = (1)(3) + (3)(-2) = -3\text{.}\)
    \(\vu\cdot\vv = (1)(3) + (3)(-4) = -9\text{.}\)
  6. The dot product of the left vectors is positive because the angle between the vectors is acute. The dot product of the middle vectors is 0 because the angle between the vectors is \(90^{\circ}\text{.}\) The dot product of the right vectors is negative because the angle between the vectors is obtuse.
