Activity 13.5.3.

In this activity, we will examine why we must be careful when using symmetry to make arguments about scalar line integral. Let \(C_1\) and \(C_2\) be the paths shown in Figure 13.5.16. We will consider the function \(f(x,y)=x\) for this activity.
(a)
Parameterize \(C_1\) and \(C_2\) as \(\vr_1(t)\) and \(\vr_2(t)\text{.}\) (It is fine to have \(0\leq t\leq 1\) for both of your parameterizations.)
Solution.
We parameterize \(C_1\) with \(\vr_1(t)= \langle -1=t,2-2t\rangle\) and parameterize \(C_2\) with \(\vr_2(t) =\langle t,t\rangle \text{.}\) Both parameterizations use parameter values in the interval \([0,1]\text{.}\) These are not the only ways to parameterize these segments.
(b)
Use Theorem 13.5.11 to compute \(\displaystyle\int_{C_1} f\, ds\) and \(\displaystyle\int_{C_2} f\, ds\text{.}\)
Solution.
We can calculate \(|{\vr_1}| = \sqrt{5}\) and \(|{\vr_2}| = \sqrt{2}\text{.}\) Thus we can evaluate our scalar line integral as
\begin{gather*} \int_{C_1} x \, ds = \int_0^1 (-1+t) \sqrt{5} \, dt = -\sqrt{5}+\frac{\sqrt{5}}{2}\\ \int_{C_2} x \, ds = \int_0^1 (t) \sqrt{2} \, dt = \frac{\sqrt{2}}{2} \end{gather*}
(c)
As with line integrals of vector fields, we have that \(\displaystyle\int_{C_1+C_2} f\, ds = \int_{C_1} f\, ds + \int_{C_2} f\, ds\text{.}\) Use this property to compute \(\displaystyle\int_{C_1+C_2} f\, ds\text{.}\)
Solution.
We can evaluate our scalar line integral as
\begin{align*} \int_{C_1+C_2} x \, ds \amp = \int_{C_1} x \, ds +\int_{C_2} x \, ds\\ \amp = \int_0^1 (-1+t) \sqrt{5} \, dt + \int_0^1 (t) \sqrt{2} \, dt\\ \amp = -\sqrt{5}+\frac{\sqrt{5}}{2}+\frac{\sqrt{2}}{2} \neq 0 \end{align*}
In the calculation above, you can see how the difference in displacements manifests differently in the two integrals (as \(\sqrt{5}\) and \(\sqrt{2}\)) which makes the final result not quite cancel out to zero.
